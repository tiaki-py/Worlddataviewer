# World Data Viewer
This application is made public with the knowledge that this school project has the capability to help people doing research on socioeconomic topics.
There is no guarantee that this repository will be maintained or updated further beyond the initial upload, but if so, there will be nothing behind it but personal dedication.

## Features
As of release v0.1-beta, the application includes the following functionality:
- Static GDP, population, language, geographic, and export data on 195 UN member and observer states
- Modern UI written in PyQt5
- Builds with Pyinstaller as single-file standalone executables and zip files
Contributors, pull requests, and questions are welcome.
